# Readify
Readify merupakan paltform membaca novel gratis dan berbayar yang dapat digunakan oleh semua orang baik hanya membaca maupun menulis novel, dan juga menjual setiap episode novel yang dibuat.

## Spesifikasi
1. PHP 8.0^
2. MySQL 5.4^
